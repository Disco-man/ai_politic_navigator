# AI Political Navigator

An interactive web application that helps people navigate the complex world of 21st century politics through AI-powered insights, interactive maps, and comprehensive country information.

To run the project:
1)Download zip

2)Unpack in any folder

3)Open terminal (Win+R then type 'cmd')

4)Example: cd C:\Users\Lenovo\Desktop\folder_name\frontend (Go to the frontend folder)

5)npm install

6)click START_DESKTOP_APP

7)type number '1'
